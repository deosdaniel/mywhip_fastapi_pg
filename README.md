## Setup Instructions


   ```bash
   pip install -r requirements.txt
   ```

---

## Running the Application

1. **Start the FastAPI Server**

   Run the following command to start the development server:

   ```bash
   fastapi dev src/
   ```

   The API will be available at `http://127.0.0.1:8000`.
